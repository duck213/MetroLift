먼저 전체적인 화면은 Android, iOS와 같이 모바일 UI/UX 기준입니다.

메인 화면에는 현재 위치정보를 기반으로 Google Maps의 지도가 펼쳐져야 함.

메인 화면 헤더(Header)에는 가운데 MetroLift라는 글씨와 그 글씨를 기준으로 왼쪽에 지구본, 오른쪽에 벨 모양이 표시되어야 함.

화면 하단에는 출발, 도착 정보를 입력해야 하며 이는 Searchable Select으로 검색 필요. 검색 후보는 한국 수도권 전철 기반. 원하는 키워드를 입력하면 확인(primary action, solid button),취소 버튼으로 선택할 수 있어야 함.

출발지 입력이 완료되면 현재 위치에서 가장 가까운 지하철 역의 엘리베이터 위치가 지도 위에 표시됨. 출발지가 입력 완료된 상태에서 도착지를 검색할 수 있어야 함. 도착지가 입력 완료되면 출발지, 도착지 정보가 지도 위에 표시되어야 함.

출발지와 도착지 정보 입력이 완료되면 경로안내가 완료된 지하철 역의 정보가 지도 위에 표시되어야 함.
